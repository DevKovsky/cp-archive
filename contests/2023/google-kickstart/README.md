# Google Kick Start 2023

이 폴더는 **2023년 Google Kick Start**에서 다룬 문제들의 풀이를 정리한 공간입니다.  
Kick Start 특유의 대규모 입력, 조합론·기하·그리디 중심 문제 스타일을 반영하여  
깔끔하고 직관적인 구현에 집중했습니다.

팀 구성:

## 📌 포함된 문제 요약

### **A — Reordered Anagrams Count**
문자열 S가 주어졌을 때, 문자를 재배열했을 때 **같은 문자가 인접하지 않는 순열의 개수**를 구하는 문제입니다.  
핵심 포인트:
- 최대 빈도수 검증(불가능성 체크)
- 중복 문자를 가진 순열 계산
- 팩토리얼 / 모듈러 역원 / 조합론

### **B — Minimum Circles to Cover Points**
1차원 좌표 위의 점들을 최소 개수의 원(구간)으로 덮는 문제입니다.  
Kick Start에서 자주 나오는 **정렬 + 그리디** 조합 문제 형태입니다.  
핵심 포인트:
- 점 정렬
- 가장 멀리 커버할 수 있는 지점을 기준으로 탐욕적 선택
- 큰 범위의 좌표 처리

---

## 🚀 복잡도 요약

| 문제 | 시간 복잡도 | 사용 알고리즘 |
|------|-------------|---------------|
| A | O(N) + 팩토리얼 사전 계산 | 조합론, 모듈러 연산 |
| B | O(N log N) | 정렬, 그리디 |
