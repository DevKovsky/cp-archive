이 폴더는 **2023 ICPC Asia Regional**에서 다룬 문제들에 대한 풀이를 정리한 공간입니다.

## 📌 포함된 문제 요약

### **A — Multi-source Dijkstra**
여러 개의 시작점(기지 도시)이 주어지고, 각 도시에는 고유의 '위험도'가 존재합니다.  
간선을 지날 때마다 도로 가중치 + 다음 도시의 위험도가 더해지는 형태로 최단 경로를 구하는 문제입니다.  
핵심 포인트:
- 다중 시작점 Dijkstra
- 1e18 수준의 긴 정수 처리
- 큰 입력 최적화

### **B — Longest Path in DAG**
N개의 작업과 선행 순서가 주어진 DAG 위에서 전체 작업을 완료하는 데 필요한 최소 시간을 구하는 문제입니다.  
핵심 포인트:
- 위상정렬
- DAG DP (longest path)
- 병렬 처리 가능성 모델링

---

## 🚀 복잡도 요약

| 문제 | 시간 복잡도 | 핵심 기술 |
|------|-------------|-----------|
| A | O((N+M) log N) | 다중 시작점 Dijkstra |
| B | O(N+M) | DAG DP, topological sort |
